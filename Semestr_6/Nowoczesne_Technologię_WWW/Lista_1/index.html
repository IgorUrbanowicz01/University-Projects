Meta Heuristic Algorithms

Metaheuristic algorithms are a type of optimization algorithm that is used to find near-optimal solutions to complex problems. Unlike traditional optimization algorithms, metaheuristic algorithms do not require an exact mathematical model of the problem being solved, and are capable of handling a wide range of problem types and structures.

The Traveling Salesman Problem (TSP) is a classic optimization problem in computer science. It involves finding the shortest possible route that visits each of a given set of cities exactly once and returns to the starting city. In order to solve the TSP I've implemented several metaheuristic algorithms, such as Genetic Algorithm, Tabu Search, Random k, and Opt-2. Genetic Algorithm is an optimization algorithm inspired by the process of natural selection, which uses a combination of selection, crossover, and mutation to generate new candidate solutions. Tabu Search is a local search-based algorithm that avoids revisiting recently visited solutions. Random k is a simple random algorithm that selects k random solutions and returns the best one. Opt-2 is a heuristic algorithm that attempts to improve an existing solution by swapping two edges. For all of the alorythms metioned above I used Python and the TSP librarie wich helps working with instances of the problem as they can be presented in a vaiarity of difrent ways.
